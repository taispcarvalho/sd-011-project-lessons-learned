Desde o primeiro momento percebi que a Trybe era diferenciada, logo nas primeiras semanas este sentimento foi ficando cada vez mais nítido. Toda a atenção e foco no aluno, no aprendizado, fazem toda a diferença. É um curso puxado, difícil, mas todo o preparo do time Trybe e as excelentes pessoas instrutoras fazem desta jornada mais leve. Foram apenas 3 semanas, mas já deu pra ver o que nos aguarda! Só tenho a agradecer!

Em apenas três semanas de curso consegui compreender conceitos tanto de hard skills quanto de soft skills, contando com o apoio e atenção de exelentes profissionais

  1. Fundamentos Unix
  2. Bash
  3. Git
  4. GitHub
  5. Inteligência Emocional
  6. Como identificar gatilhos emocionais
  7. Noções de como otimizar trabalhos em grupo
  8. HTML
  9. HTML semântico
  10. CSS
