Nas ultimas semanas tive aulas de programação com o professor Gabriel Oliva, aprendi muito sobre HTML, CSS e até sobre o github. No github finalmente aprendi da forma certa como criar um repositório e como funcionam as branchs. Já no HTML e CSS tudo foi novidade pra mim e criar os meus primeiros "esqueletos" de site está sendo uma experiencia incrível!

Aprendi a criar pastas e diretórios pelo terminal e até a procurar arquivos por todo o computador com apenas um comando. Também vimos sobre permissões de usuários e modos de bloquear acesso, podendo até mesmo tornar um arquivo, se necessário, invisível e inacessível.
