O aprendizado aqui na Trybe se fez presente desde a preparação para o processo seletivo, momento em que estudei lógica de programação com JavaScript, também revisei algoritmos e estrutura de dados que foram aplicados no teste prático, em relação ao teste de lógica me preparei com silogismo e teoria dos conjuntos.
Após o início do curso, aprendi a utilizar o terminal para criar e navegar pelos diretórios. Posteriormente, comecei a versionar o código usando o Git e o repositório remoto do Github, e a criar os primeiros sites estáticos usando HTML e CSS.
Desde o momento que iniciei os meus estudos na Trybe, os maiores desafios têm sido a utilização do Github e a estilização das páginas HTML por meio do CSS.
