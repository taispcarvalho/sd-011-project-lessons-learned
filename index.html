Trybe

Sobre o curso da Trybe

Com menos de 3 semanas de curso conseguimos aprender um pouco sobre Unix e comandos uteis para se usar no terminal, Git e seus comandos (add, commit, push, pull, branch, checkout, merge), GitHub e como abrir PR (Pull Request) e fazer CR (Code Review), HTML e sua estrutura e semantismo e CSS com suas estilizacoes via tag, classe e id.

Durante o curso sao propostos diversos exercicios e projetos para fixar os conteudos aprendidos. E para cada projeto temos tambem requisitos bonus para desafiar nossos conhecimentos e instigar o uso da soft skill Aprender a Aprender.
