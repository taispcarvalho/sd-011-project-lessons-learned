Primeiros Passos

Se passaram 3 blocos do módulo de Fundamentos de Desenvolvimento Web e já fui apresentado a conceitos importantes que vão guiar toda minha formação e além.

Tendo Compreensão do funcionamento do sistema Unix, utilização do terminal de forma confortável, conhecendo os principais comandos do Bash e conhecendo o funcionamento da internet.

Utilizar o Git, ir do git init ao git push, permitindo guardar meus projetos em um repositório remoto - Github e Fazer Code Review.
