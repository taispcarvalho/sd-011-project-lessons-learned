Sobre o Aprendizado

Desde o inicio do curso fomos nos desenvolvendo em Unix, Bash, Git, HTML e CSS de forma gradual e satisfatória. Aprendemos comandos com o git como o add, commit, push, pull, branch e merge. Em HTML, tags como header, nav, aside, footer, article. E em CSS, vimos display (block, inline, inline-block) além de outros para estilização da página.

De Unix, Bash, Git, HTML e CSS até este projeto. Venha conosco e descubra como foi essa incrivel tragetória.
