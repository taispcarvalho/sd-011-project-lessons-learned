Desenvolvimento web é o termo utilizado para descrever o desenvolvimento de sites. O profissional que trabalha desenvolvendo websites pode ser um web designer (desenvolvedor do layout - Frontend ou UI dev) ou um web developer (desenvolvedor de sistemas - Backend). Bem como o Full Slave Full Stack Developer, que cuida tanto de Back quanto de Front.
Iremos aprender diversas ferramentas para o desenvolvimento web, linguagens como HTML, CSS, Javascript, SQL, etc bem como frameworks e banco de dados.
